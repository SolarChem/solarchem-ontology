The SolarChem Ontology: Photocatalysis Experiments: Description back to ToC

This module extends the SolarChem ontology with several concepts. First, it adds a subclass of `solar:ExperimentExecution` to denote photocatalysis experiments, `solarpc:PhotocatalysisExperiment`. Then, it adds three types of the roles that an input chemical substance can play in the experiment (i.e. dye, reductant and support), and some additional conversion metrics (i.e. quantum yield, photonic efficiency and apparent quantum yield).
