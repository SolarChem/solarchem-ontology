Introduction back to ToC

Ontology for the representation of three types of catalysis experiments described in scientific papers, namely, phtocatalysis, electrocatalysis and photoelectrocatalysis. The experiments description includes how catalysis experiment has been performed; under which conditions, what input products were used and the role in the chemical reaction, and what have been the results in order to analyze and reproduce the experiments mentioned in the articles. This core ontology provides the essencial concepts for describing these experiments, and the shared concepts among at least two types of catalysis experiments. This ontology is extended by three modules (one per type of catalysis) to provide specific concepts for each type of catalysis.
