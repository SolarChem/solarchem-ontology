The SolarChem Ontology: Electrocatalysis Experiments: Description back to ToC

This module extends the core SolarChem Ontology in two parts. First, it adds a subclass of `sc:ExperimentExecution` to denote electrocatalysis experiments, `scec:ElectrocatalysisExperiment`. Then, it adds to the hierarchy of electrochemical metrics of the outputs two concepts, turn over frequency (`scec:TurnOverFrequency`) and Tafel slope (`scec:TafelSlope`).
