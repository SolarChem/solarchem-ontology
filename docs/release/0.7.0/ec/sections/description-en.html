The SolarChem Ontology: Electrocatalysis Experiments: Description back to ToC

This module extends the core SolarChem Ontology in two parts. First, it adds a subclass of `sc:ExperimentExecution` to denote electrocatalysis experiments, `solarec:ElectrocatalysisExperiment`. Then, it adds to the hierarchy of electrochemical metrics of the outputs two concepts, turn over frequency (`solarec:TurnOverFrequency`) and Tafel slope (`solarec:TafelSlope`).
