The SolarChem Ontology: Photoelectrocatalysis Experiments: Description back to ToC

This module extends the SolarChem ontology with several concepts. First, it adds a subclass of `sc:ExperimentExecution` to denote photoelectrocatalysis experiments, `scpec:PhotoelectrocatalysisExperiment`. Then, it adds several types of the roles that an input chemical substance can play in the experiment (e.g. photoabsorber, photocatalyst, substrate). The output description is extended with a dedicated metric, `scpec:QEMetric`, and additional types of electrochemical metrics (e.g. onset) and electrochemical configurations (e.g. PEC, PV-PEC).
