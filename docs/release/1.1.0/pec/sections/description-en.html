The SolarChem Ontology: Photoelectrocatalysis Experiments: Description back to ToC

This module extends the SolarChem ontology with several concepts. First, it adds a subclass of `solar:ExperimentExecution` to denote photoelectrocatalysis experiments, `solarpec:PhotoelectrocatalysisExperiment`. Then, it adds several types of material transformation roles that an input chemical substance can play in the experiment execution (e.g. photoabsorber, photocatalyst, substrate). The observable properties are extended with a dedicated metric, `solarpec:QuantumEfficiencyMetric`, and additional types of electrochemical metrics (e.g. onset). Likewise, there are dedicated conditions for electrochemical configurations (e.g. PEC, PV-PEC).
